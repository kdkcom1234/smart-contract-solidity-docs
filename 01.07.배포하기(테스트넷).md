### 폴리곤 테스트넷에 배포

```javascript
require("@nomicfoundation/hardhat-toolbox");
require("dotenv").config();

/** @type import('hardhat/config').HardhatUserConfig */
module.exports = {
  solidity: "0.8.24",
  networks: {
    amoy: {
      url: process.env.AMOY_RPC_URL, // 환경 변수에서 RPC URL을 읽어옵니다.
      accounts: [process.env.PRIVATE_KEY], // 개인 키를 환경 변수에서 읽어옵니다.
    },
  },
};
```

배포가 성공적으로 완료되면, 스마트 컨트랙트의 주소를 기록해둡니다.

```shell
npx hardhat ignition deploy ./ignition/modules/Greeting.js --network amoy
# 0xcb908ba336002b6cea8786b31b85d10e3fc853dc
# https://www.oklink.com/ko/amoy/address/0xcb908ba336002b6cea8786b31b85d10e3fc853dc
```
